## How to Take Screenshots

### For Development Server:

1. Run `npm run dev`
2. Open `http://localhost:5173` in your browser
3. Take a screenshot
4. Save as `screenshots/dev-server.png`

### For Vitest UI:

1. Run `npm run test:ui`
2. Wait for the browser to open automatically
3. Take a screenshot showing all tests passing
4. Save as `screenshots/vitest-ui.png`
